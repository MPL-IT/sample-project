How to Build the Documentation
------------------------------

In the `docs` folder, Mac or Linux users type

.. code-block::

    make html


and Windows users execute

.. code-block::

    make.bat html


To trigger the build of the documentation. 

After that open `docs/build/index.html`
in your browser.
